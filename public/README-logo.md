# Logo PNG de Chikiglam

Para que funcione correctamente, guarda la imagen del logo como "logo.png" en esta ubicación.

El logo debe tener fondo transparente para mejor resultado en el header.

Dimensiones recomendadas:
- Ancho: 200-300px  
- Alto: 60-100px
- Formato: PNG con transparencia
- DPI: 72-150 para web

Una vez guardado como logo.png, se verá automáticamente en el sitio.

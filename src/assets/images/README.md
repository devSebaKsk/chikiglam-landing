# Carpeta de Imágenes

Aquí van las imágenes del proyecto:

## Estructura recomendada:
- `hero/` - Imágenes para la sección hero
- `services/` - Imágenes de servicios
- `team/` - Fotos del equipo
- `gallery/` - Galería de trabajos
- `testimonials/` - Fotos de clientes (con permiso)

## Formatos recomendados:
- **WebP** para mejor compresión
- **JPG** para fotografías
- **PNG** para imágenes con transparencia
- **SVG** para iconos y logos

## Optimización:
- Usa herramientas como TinyPNG o Squoosh.app
- Implementa lazy loading donde sea necesario
- Considera usar CDN para mejor performance

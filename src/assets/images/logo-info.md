# Logo de Chikiglam

Este archivo contiene información sobre el logo de la marca.

Para usar el logo:
1. Coloca el archivo logo.svg o logo.png en esta carpeta
2. Importa desde los componentes usando: import logo from '../../assets/images/logo.svg'

## Versiones recomendadas:
- logo.svg - Versión vectorial (recomendada)
- logo.png - Versión rasterizada de alta calidad
- logo-white.svg - Versión en blanco para fondos oscuros
- logo-icon.svg - Solo el ícono CG sin texto
